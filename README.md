# 💵 People on Banknotes: Exploratory Data Analysis

## 📖 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a dataset of historical figures featured on banknotes from countries around the world. Using Python and data visualization techniques, the analysis explores patterns in gender representation, occupations, countries, currencies, and the timeline of banknote appearances.

The objective is to gain insights into how different nations choose notable personalities to represent their culture, history, and identity on currency.

---

## 🎯 Objectives

- Explore the dataset using Python and Pandas.
- Clean and prepare the data for analysis.
- Analyze gender representation on banknotes.
- Identify the most common occupations featured.
- Compare countries based on the number of historical figures represented.
- Study the timeline of first appearances on banknotes.
- Create informative visualizations to communicate findings.

---

## 🔍 Key Findings

- Male historical figures are featured much more frequently than female figures.
- Political leaders, national heroes, and cultural icons are among the most commonly represented occupations.
- Representation varies considerably across countries.
- Female representation on banknotes remains comparatively limited.
- Banknotes often celebrate individuals with significant historical, political, scientific, or cultural contributions.
